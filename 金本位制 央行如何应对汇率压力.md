[[只有 Dishoarding 是解决流动性危机最稳定可靠的方式]]
# 场景

![image.png|552](https://s1.vika.cn/space/2025/06/22/8f7c4fb8e6ce4608bceaa9adf6785cfe)

**基础场景设定**
   - 私人交易商持有本币（如美元）向央行兑换国际货币（英镑或黄金）。
   - 央行卖出黄金/英镑，买入本币 → 导致**资产负债表收缩**（储备减少、本币流动性下降）。
   - 这就是**被动货币紧缩**：并非主动调控利率，而是交易商行为自然导致货币供应减少。


# 三种应对手段

![image.png|555](https://s1.vika.cn/space/2025/06/22/8b477a707483477396edb1e7cb6133df)

   **(1) 直接消耗储备（不可持续）**
   - 持续卖出黄金或英镑储备，但储备有限。

![image.png|554](https://s1.vika.cn/space/2025/06/22/d97a39332c184ae0840b8b7411cb2599)

   **(2) 出售资产换取储备（如国债）**
   - 央行抛售国债（T-bills）换取国际货币（英镑/黄金）
   - **副作用**：国债价格下跌 → 利率上升（紧缩效应加剧）。

![image.png|552](https://s1.vika.cn/space/2025/06/22/beb1541910d447169611c6066be24ba0)

   **(3) 向其他央行借入储备**
   - 例如向英格兰银行借款，增加外汇储备（负债端记为"借入储备"）。
   - **关键优势**：央行比私人交易商多一层融资渠道。


# "Reserve储备"的本质 

   - **国际视角**：此处"储备"指黄金或英格兰银行发行的英镑（国际储备货币）。
   - **国内视角**：商业银行的"储备"是本币存款或央行票据，但在此模型中，央行自身是本币发行者，需用**外部硬通货**应对汇率压力。


# 课程核心视角：关注流动性


> [!important]
> 通过**交易商模型**还原真实市场机制，而非抽象均衡理论。


**金本位制的实际运行逻辑**
   - 传统理论常假设汇率严格贴合铸币平价（X/Y），但现实中存在**黄金输送点**（±Δ）形成的波动区间：
     - 汇率波动至边界时，黄金流动强制调节。
     - **价差结构**：
       - **内部价差**：外汇交易商的买卖报价（市场实际成交区间）
       - **外部价差**：黄金输送点决定的硬性边界（Δ=黄金运输成本）
   - 忽略价差等于忽略**流动性提供者（交易商）的盈利动机**，而这是市场运转的核心。
