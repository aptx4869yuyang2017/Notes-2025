蒙代尔将第二阶段定义为**国家宏观经济管理与固定汇率体系的对抗**，核心矛盾在于：

#  各国政策自主性与固定汇率的冲突


- 每个国家财政部和央行都想推行利于本国的政策
- 但布雷顿森林体系要求维持固定汇率
- 除非有“矛盾吸收者”（美国本可承担此角色但未履行）

# 凯恩斯的“班科计划”（Bancor Plan）


![image.png](https://s1.vika.cn/space/2025/06/21/57946f387f0f46909be94b958ca46a38)



   - **设计目标**：建立全球央行（国际清算联盟）和超主权货币“班科 Bancor”
     - 赤字国和盈余国通过班科结算
     - 国际货币数量可动态调整（类似央行通过扩表解决流动性问题）
     - 消除金本位的通货紧缩倾向，**对称调整责任**（盈余国也需承担调整压力），削弱仅约束逆差国、却放任顺差国的生存压力
	     - 通过设置激励措施迫使其将盈余用于消费（购买逆差国商品或资本）而非储蓄
   - **美国反对原因**：
     - 实质是要求美国长期向欧洲提供贷款
     - 凯恩斯提议对存款（盈余国持有资金）征收利息，变相惩罚美国顺差
     - 被美国视为“无控制权的强制放贷机制”



# IMF与SDR（特别提款权）


![image.png](https://s1.vika.cn/space/2025/06/21/0cfa21cc024a4e96bdfec6985f31bad4)


   - **运作逻辑**：
     - 成员国向IMF缴纳黄金和本币，换取SDR（固定数量的“纸黄金”）
     - 赤字国用SDR支付，但SDR总量不变（仅转移所有权）
     - 耗尽SDR后需向IMF借款，接受附加条件（紧缩政策等）
   - **根本缺陷**：
     - 维持调整不对称性（赤字国独自承担压力）
     - 国际储备总量僵化，无法动态适应需求


# **美元的实际角色**


   **查尔斯·金德尔伯格理论**：
 - 虽然IMF的SDR供给有限，但美元在货币层级中的扩张暂时缓解了国际储备短缺
 - 美国通过输出美元替代黄金，成为实际上的“全球流动性提供者”



# 关键对比

| **班科计划**    | **IMF体系**    |
| ----------- | ------------ |
| 强调弹性        | 强调纪律         |
| 国际货币弹性供应    | 国际货币固定数量     |
| 盈余国与赤字国对称调整 | 仅赤字国承担调整压力   |
| 全球央行主导      | 美国实际主导（通过美元） |
