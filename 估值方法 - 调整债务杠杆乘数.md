**净负债 = 总债务 − 现金**




# 无杠杆市销率 PS


**无杠杆市销率** = （权益市场价值 + 负债净额） / 销售收入


A公司（净现金50亿）、B公司（净负债50亿），市值均为200亿，年营收100亿。

- 传统P/S：均为 200 / 100 = **2.0x**
- 无杠杆EV/Sales：  
    A公司 EV = 200 − 50 = 150 → EV/Sales = **1.5x**  
    B公司 EV = 200 + 50 = 250 → EV/Sales = **2.5x**  
    → 实际上，市场为B公司的业务多付了50%溢价，反映其更高风险或更强增长预期；而A公司被显著低估（若业务相同）。传统P/S完全掩盖了这一关键差异。


# 无杠杆价格-EBIT比率

权益市场价值 + 负债净额） / EBIT  

其中，EBIT = 息税前利润（净利润 + 利息支出 + 所得税）



假设两家公司 A 和 B：

|公司|销售收入|EBIT|股东权益市值|净负债|
|---|---|---|---|---|
|A|100|20|80|20|
|B|100|20|90|0|

- A 借了很多钱（净负债20），所以权益市值较低（80）
- B 没借钱，权益市值较高（90）

如果只看 **市销率 = 权益市值 / 销售收入**：

- A：80 / 100 = 0.8
- B：90 / 100 = 0.9 → 看起来B更贵

但这不公平！因为A用了杠杆，实际企业价值更高。

用**无杠杆市销率**：

- A：(80 + 20) / 100 = 1.0
- B：(90 + 0) / 100 = 0.9 → 实际上A更贵

所以，**无杠杆指标能更真实反映企业本身的价值**。


# 企业市净率

（权益市场价值 + 负债净额） / （权益账面价值 + 负债净额）



假设某公司如下数据：

|项目|数值（单位：亿元）|
|---|---|
|股票价格|10 元/股|
|总股本|1亿股|
|权益的账面价值（股东权益）|8 亿元|
|负债净额（有息负债）|6 亿元|

#### 第一步：计算分子（企业价值 EV）

- 权益的市场价值 = 股价 × 总股本 = 10 × 1亿 = **10 亿元**
- 加上负债净额 = 10 + 6 = **16 亿元**

#### 第二步：计算分母（账面价值）

- 权益账面价值 + 负债净额 = 8 + 6 = **14 亿元**

#### 第三步：计算企业市净率

企业市净率=1416​≈1.14
