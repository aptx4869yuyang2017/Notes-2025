# 1. 商品本位论（Chartalist/购买力平价）

![image.png|393](https://s1.vika.cn/space/2025/06/21/00a0d72a2ecf491d9892eea800f7990d)


- 核心逻辑：汇率是两国可贸易商品的相对价格
- 局限：忽视了资本流动和货币的支付媒介属性

- [[货币名目论 Chartalism]] 货币的价值来自国家认证
- 因为 [[货币数量论]] MV=PT 国家虽然能决定货币发行数量，但是不能决定货币代表的价值
- 所以推导出国家之间的货币价格比率（汇率），来自两国商品的相对价格 [[购买力平价理论 PPP]]
- 这一派的经济学家喜欢看 贸易账户/经常账户


# 2. 金属本位论（Metallist/利率平价）

[[货币金属论 Metallism]]

![image.png|409](https://s1.vika.cn/space/2025/06/21/573b5be5c6ec4aa391a3b8463431f452)


- 核心逻辑：汇率反映**两国金融资产的相对价格**
- 局限：假设完全市场流动性，忽略了信用创造过程



# 3. 货币信用观（Money View）的创新视角

![image.png|529](https://s1.vika.cn/space/2025/06/21/969645428b0b4a84a857cad60a8c2f88)


- **汇率本质**：货币与货币之间的即时兑换价格
  - 体现在做市商 Dealer 资产负债表上的买卖报价
  - 关键问题：当逆差国缺乏美元时，支付如何实现？
- **核心机制**：
  - 做市商通过即期市场创造支付流动性
  - 将汇率风险通过远期合约转移至投机者（类似货币市场的期限转换）
- **理论突破**：
  - **拒绝抽象化流动性**（与传统经济学/金融学根本差异）
  - 揭示汇率本质是**支付系统的信用定价工具**

**辩证关系**
- 前两种理论部分正确但过度简化：
	- 购买力平价侧重商品流动
	- 利率平价侧重资本流动
- 货币信用观将二者纳入**支付链条的动态过程**：
	- 即期汇率由做市商的流动性供给决定
	- 远期汇率反映风险转移成本
