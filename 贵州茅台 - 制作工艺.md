- 生产由 上市公司本部完成
- 粮耗比：1:5 
	- 1t 茅台 = 2.6t 小麦 + 2.4t 高粱
- 生产周期1年，2次投料，9次蒸煮，8次发酵，7次取酒
- 周期
	- **取酒**  一年 7次
		- 9%/ 14% / 25% / 22% / 15% / 10% / 5%  
	-  **基酒**：合并同类项，分酒质分批次，酒库陶坛3年
		- 酱香 / 醇甜 / 窖底
	- **成品酒**勾兑
		- 不掺水
		- 醇化/老熟 存放 0.5～1年
