| 序号                                       | 局限性                                                                 | 详细说明                                                        |
| ---------------------------------------- | ------------------------------------------------------------------- | ----------------------------------------------------------- |
| **1. 高度依赖长期预测（主观性强）**                    | DCF需预测未来5–10年甚至更久的自由现金流（FCF），而远期收入、成本、资本支出等极难准确估计。                  | 假设第8年营收增长5% vs 3%，终值可能相差30%以上；科技公司因技术迭代快，预测误差更大。            |
| **2. 终值（Terminal Value）占比过大且敏感**         | 在典型DCF中，终值常占总估值的60%–90%。其计算依赖永续增长率（_g_）和折现率（_r_），微小变动引发估值剧烈波动。      | 若 _g_ 从3%调至3.5%（仅+0.5%），终值可能上升20%；若 _r_ 从9%升至9.5%，估值下降约15%。 |
| **3. 违背会计配比原则（Conceptual Misalignment）** | DCF将投资支出（如建厂）计入当期现金流出，但相关收入分摊在未来多年——**成本与收益不匹配**，与会计“权责发生制”和配比原则冲突。 | 会计上可能将厂房折旧分10年摊销，而DCF直接在t=0扣1亿元现金支出，导致跨期价值归属不清。             |
| **4. 忽略非财务/战略价值**                        | DCF只量化可货币化的现金流，无法反映品牌力、用户生态、数据资产、先发优势、协同效应等无形价值。                    | 星巴克的会员体系、可口可乐的渠道控制力、特斯拉的软件生态，均难以通过DCF现金流体现。                 |
| **5. 对参数极其敏感（脆弱性高）**                     | 估值结果对折现率（WACC）、永续增长率、税率、营运资本变动等关键假设高度敏感，缺乏稳健性（Robustness）。          | 教学案例中常见：同一公司，不同分析师因WACC假设差1%，估值差异可达±25%。                    |
| **6. 不适用于负现金流或早期企业**                     | 初创公司、生物技术企业、平台型公司常多年无正向自由现金流，DCF无法直接应用（分母为负或无意义）。                   | Airbnb在2015年前持续亏损；mRNA疫苗公司早期研发投入巨大，DCF会得出负估值或严重失真。          |
