| 项目         | 债权投资      | 其他债权投资              |
| ---------- | --------- | ------------------- |
| **会计计量方式** | 摊余成本法     | 公允价值计量（OCI）         |
| **持有目的**   | 收取本金和利息   | 收取现金流 + 可能出售        |
| **收益影响**   | 利息入损益     | 利息入损益，公允价值变动入其他综合收益 |
| **减值处理**   | 【预期信用损失】  | 同样计提【预期信用损失】        |
| **处置时处理**  | 仅确认实际利息收益 | 将累积的 OCI 调整至“投资收益”  |
