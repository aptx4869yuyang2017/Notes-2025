# 难点


这段文字主要讨论了**资产负债表所列资产在衡量企业内在价值时的局限性**，并指出了几个关键问题：

1. **资产市场价值难以获取**：  
    资产负债表上的资产通常不是经常交易的，因此很难找到它们的市场价格。
    
2. **市场价值不等于内在价值**：  
    即使能获得市场价值，这些价格也不一定反映资产的真实内在价值，因为市场可能不完善，或市场价值不能体现资产在企业持续经营中的实际贡献。
    
3. **使用情境影响资产价值**：  
    同一资产在不同用途下创造的价值不同。例如，一栋建筑用于生产电脑和用于仓储，其价值差异显著，说明市场价值（如重置成本或清算价）未必代表其在企业运营中的真实价值。
    
4. **无形资产被忽略且难以估值**：  
    资产负债表往往忽略品牌、知识等无形资产，而这些资产对企业价值至关重要。但因其缺乏客观交易市场，难以准确估价。
    
5. **协同效应导致整体价值难估**：  
    即使能分别估算每项资产的价值，这些资产组合后的整体价值（即“协同作用”产生的价值）也难以确定。企业通过整合资源创造出超越单项资产总和的价值，这种综合价值是估值中的难点。
    



# 案例

- **以资产为基础的估值方法**：该方法有时适用于以自然资源为主要资产的企业。
- **资源型企业（asset-based companies）**：指主要资产为油田、矿藏、林地等自然资源的企业。
- **估值方式**：对这类企业，通常通过估算其自然资源（如石油、矿产、林木）的储量或可开采量，并结合资源变现能力来评估企业价值。
