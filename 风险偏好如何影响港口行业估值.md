**核心逻辑：**
风险偏好 ↗ → 港口估值 ↗
风险偏好 ↘ → 港口估值 ↘

### **影响机制（1）：通过绝对估值模型（DCF）**

1. **折现率（r）渠道**
    
    - **路径**：风险偏好下降 → 投资者要求更高的风险溢价 → 资本成本（WACC）上升 → **折现率（r）提高** → 未来现金流现值减少 → 估值下降。
    - **特点**：港口业资产重、周期长，对折现率变化**极度敏感**。
        
2. **现金流（CF）预期渠道**
    
    - **路径**：风险偏好下降 → 市场预期经济衰退、全球贸易萎缩 → 港口**吞吐量和费率增长预期下调** → 未来现金流（CF）预测值减少 → 估值下降。
        

### **影响机制（2）：通过相对估值倍数（Multiples）**

- **路径**：风险偏好下降 → 市场整体风险资产遭抛售 → 投资者愿意为每单位盈利（EPS）或净资产（BPS）支付的价格降低 → **市盈率（P/E）、市净率（P/B）等估值倍数收缩** → 估值下降。
    
- **特点**：港口作为**强周期板块**，其估值倍数的波动幅度通常大于市场平均水平。

---

**一句话总结：市场情绪（风险偏好）通过影响投资者对港口未来盈利的信心和要求的回报率，直接决定其估值高低。**
