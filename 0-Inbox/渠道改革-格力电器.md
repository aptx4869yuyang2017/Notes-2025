# 控股销售公司  2000s中期


![image.png|341](https://s1.vika.cn/space/2025/08/25/2a80d72972294c16b63c9761079407fc)

**核心策略：**
- **控股销售公司**：格力总部出资收购各地销售公司的股份，实现绝对控股。
- **渠道垂直化**：结构变为“总部 → 控股销售公司 → 经销商”。

**目标：**
- 削弱地方诸侯权力，强化中央集权。
- 应对国美、苏宁等大连锁渠道的挑战。

**结果：**
- 总部控制力达到顶峰，形成高效协同的垂直体系。
- 即便与传统连锁卖场决裂，依靠自有专卖店体系（超3万家）仍能领先。

# 初步线上化 2010s中后期

**核心策略：**

- **防御性试水**：在天猫、京东开设旗舰店，但产品线与线下区分。
- **自建平台**：上线“格力商城”，但影响力有限。

**目标：**
- 应对电商崛起，尝试线上渠道。

**结果：**
- 线上发展缓慢，成效有限。
- 线上线下冲突加剧，2019年线上份额被奥克斯超越，危机显现。

# 新零售革新 2020-至今


[[代理商转型-格力电器]]
[[品牌商争夺渠道控制权的方法]]

**核心策略：**

- **董明珠直播**：董事长亲自直播带货，统一改革思想，证明线上潜力。
- **渠道扁平化**：取消省级代理，经销商通过 **“格力董明珠店”** 直接向总部打款提货。
- **角色转变**：原有销售公司从“批发商”转为“服务商”，赚取服务佣金。
- **线上线下融合**：线上下单，线下经销商提供配送、安装与服务。
    

**目标：**

- 彻底拥抱线上，实现渠道扁平化，去库存降成本。
- 直达用户，获取数据，向数字化转型。
    

**结果与挑战：**

- **成效**：线上收入占比大幅提升，渠道效率提高。
- **挑战**：改革阵痛剧烈，引发部分经销商反弹（如河北事件），转型仍在进行中。
