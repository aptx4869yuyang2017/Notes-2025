#### **1. 基本信息**  
- **全称**：中国外汇交易中心（China Foreign Exchange Trade System, CFETS）  
- **成立时间**：1994年（配合中国汇率并轨改革）  
- **监管机构**：中国人民银行（PBOC）  
- **官网**：[www.chinamoney.com.cn](http://www.chinamoney.com.cn)  

#### **2. 核心职能**  
- 运营银行间**外汇、货币、债券**市场交易系统  
- 发布**人民币汇率中间价**（如美元/人民币基准价）  
- 提供交易、清算、信息服务，维护市场稳定  

#### **3. 主要市场与产品**  
| **市场**  | **主要交易品种**          |
| ------- | ------------------- |
| **外汇**  | 即期、远期、掉期、期权（人民币兑外币） |
| **货币**  | 同业拆借（SHIBOR基准）、回购协议 |
| **债券**  | 国债、政策性金融债、信用债       |
| **衍生品** | 利率互换、标准债券远期         |

#### **4. 关键机制**  
- **汇率定价**：参考“上一日收盘价+一篮子货币汇率变化”  
- **交易模式**：询价、竞价、做市商制度  
- **对外开放**：  
  - **债券通**（2017年，连接境外投资者与中国债市）  
  - **外汇市场准入**（允许境外央行、机构参与）  

#### **5. 重要性**  
- 中国金融市场的核心基础设施  
- 推动**人民币汇率市场化**和**资本项目开放**  
- 促进国际投资者参与（如通过“债券通”）
