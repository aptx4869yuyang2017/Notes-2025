# IPO的主要流程

1. **前期准备**
    - 选择中介机构：投行（承销商）、律师事务所、会计师事务所等。
    - 财务审计与合规审查：确保公司符合上市条件（如盈利、股权清晰等）。
2. **提交申请**
    - 向监管机构（如中国证监会、美国SEC）提交招股说明书（Prospectus），披露财务、风险、业务模式等信息。
3. **审核与路演**
    - 监管机构审核（可能需数月到数年）。
    - 通过后，公司管理层与投行进行路演（Roadshow），向潜在投资者推介股票。
4. **定价与发行**
    - 投行通过市场调研（与机构客户、对冲基金沟通）确定发行价。
        - 例如：公司当前估值8000万美元（400万股），每股20美元，新发行1000万股融资2亿美元。
    - **目标**：定价需平衡公司融资需求与市场接受度，避免首日破发（股价下跌）。
5. **上市交易**
    - 股票在交易所（如纽交所、纳斯达克、港交所、沪深交易所）挂牌交易。

# 一级市场 vs 二级市场

![image.png|415](https://s1.vika.cn/space/2025/07/19/3635e913bcae4dab8563526fc02060b4)

- **IPO（一级市场）**：投资者直接购买新股，资金流入公司。
- **二级市场**：后续股票交易（如纳斯达克）仅是股东之间的买卖，资金不流向公司。
