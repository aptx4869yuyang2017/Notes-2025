[[海曼 明斯基 Hyman Minsky]] 提出  [[金融不稳定性假说]]  前的分析的起点

- 资本主义本质上是一种以金融为核心的经济体系，其特有的行为属性主要体现在金融活动对整体经济运行方式的塑造和影响上。
- **广义的"金融机构"**：所有经济主体（家庭/企业/政府）均需管理现金流，本质上都具有金融属性。
    - **机遇**：信贷使当前不可能的未来计划成为可能；
    - **风险**：债务承诺可能因计划失败转化为长期负担。

# 信用的内在不稳定性

### **信用不稳定性 极简总结**  

1. **信用双刃剑**  
   - 借钱消费 → 别人收入增加 → 经济变热  
   - 还债省钱 → 别人收入减少 → 经济变冷  

2. **两种观点**  
   - **霍特里**：信用容易失控，央行要提前踩刹车防泡沫  
   - **熊彼特**：信用推动创新，管太死会阻碍进步  

3. **泡沫怎么来的**  
   - 借钱买资产 → 资产涨价 → 能借更多钱 → 继续买买买（循环加速）  
   - 央行加息也拦不住，除非利率高到离谱（但会误伤经济）  

4. **泡沫怎么炸的**  
   - 利率涨 → 有人还不起钱 → 甩卖资产 → 价格暴跌 → 更多人破产（连锁反应）  

5. **央行很头疼**  
   - 涨潮时管不住（大家绕开监管）  
   - 退潮时必须救（但救多了又养懒汉）  

**核心**：信用像油门和刹车装在一起，央行得一边开一边修车。
