# Swap 卖方的操作像是 Repo

![image.png](https://s1.vika.cn/space/2025/08/21/56d3e2cb4b2f4138a42d294fe6a40bb1)


我们知道：

- **Repo（回购）** 的经济含义：
    - 投资者通过回购市场 **融资买入债券**，获得债券的利息收入，同时要支付一个浮动的融资成本（通常是隔夜利率/短期利率）。
    - 现金流结构就是 **收固定票息，付浮动利率（资金成本）**。
这正好对应了 **IRS 卖方（浮动付、固定收）** 的头寸特征：
- 他就像是 **融资买了一只固定利率债券**，票息收入是固定的，而融资成本是浮动利率。
- 因此，从经济实质上，IRS 卖方 = 借钱买债券（repo 的头寸）。
