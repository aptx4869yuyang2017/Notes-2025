### **普惠小微贷款高速增长总结（卡片笔记）**  

---

#### **📊 现状概览**  
- **规模**：35.57万亿元（2025Q2），7年增长5倍  
- **占比**：占全部贷款13%（2017年仅6%）  
- **增速**：同比+12.3%，快于整体贷款增速（+5.2pct）  
- **统计口径**：单户授信<1000万的小微企业/个体户/企业主经营贷  

---

#### **🚀 增长核心驱动力**  
**1. 政策强监管驱动**  
- 国有大行"一票否决"式考核，倒逼放量  
- 逆周期调节要求银行履行社会责任  

**2. 企业生存依赖**  
- 小微企业现金流为负→靠续贷维持经营  
- 断贷可能引发倒闭潮（就业+社会稳定）  

**3. 利率套利空间**  
- 普惠利率3.69% vs 房贷>4% →催生非真实需求  

---

#### **⚠️ 风险警示**  
| 风险类型     | 具体表现            |     |
| -------- | --------------- | --- |
| **资产质量** | 不良率持续攀升         |     |
| **市场乱象** | 中介伪造材料/银行审核放松   |     |
| **政策依赖** | 真实需求不足，增长可持续性存疑 |     |

---

#### **🛠️ 监管应对**  
- 多地金监局风险提示（深圳/天津/广东等）  
- 银行自查清理违规中介合作  
- **核心矛盾**：政治任务vs风险控制的平衡  

---

#### **💡 关键洞察**  
- **增长本质**：政策驱动型增长＞市场自然需求  
- **两难困境**：  
  - 不放贷→小微企业倒闭  
  - 过度放贷→不良率恶化  
- **未来焦点**：从"规模导向"转向"质量导向"
