准备金的分类与美联储的调控**

1. **总准备金（TR）**  
   - 银行存在美联储的所有钱，分为两部分：  
     - **法定准备金（RR）**：必须持有的最低金额（由存款准备金率决定）。  
     - **超额准备金（ER）**：超出法定要求的闲置资金，可自由使用（如放贷）。  

2. **按来源分类**  
   - **非借入准备金（NBR）**：银行通过市场（如客户存款、美联储买国债）自然获得的资金。  
   - **借入准备金（BR）**：银行从美联储贴现窗口借的钱（需付利息）。  
   - **关系**：  
     $\text{总准备金（TR）} = \text{非借入（NBR）} + \text{借入（BR）}$
     

3. **自由准备金（Free Reserves）**  
   - **公式**：`自由准备金 = 超额准备金（ER） - 借入准备金（BR）`  
   - **意义**：反映银行体系的流动性松紧：  
     - **正数**：银行资金充裕，市场利率可能下降。  
     - **负数**：银行依赖美联储借款，市场利率可能上升。  

4. **美联储如何调控？**  
   - **控制利率**：通过调节非借入准备金（如买卖国债）影响联邦基金利率（短期利率）。  
   - **控制数量**（罕见）：直接限制准备金总量（如沃尔克1980年代抗通胀，但副作用大）。  

### **一句话核心**  
美联储通过调整**准备金来源（借入vs非借入）** 和 **超额准备金水平**，间接控制市场利率和信贷规模，最终影响经济。
自由准备金是观察银行流动性的关键指标。
