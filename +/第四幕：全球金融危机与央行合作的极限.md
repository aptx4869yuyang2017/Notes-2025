# 危机前

![image.png](https://s1.vika.cn/space/2025/06/21/0df9c74a59484bbca26c0b789b650f4d)

1. **欧洲银行的美元游戏**
    - 瑞银等欧洲银行大量购入美国房贷债券（MBS）
    - 通过短期美元借款（如货币市场基金融资）进行长期投资，赚取利差
2. **脆弱性根源**
    - 这些美元融资来自离岸市场（不受美联储监管）
    - 资金提供者（如外国央行）刻意避开美国司法管辖


# 危机中


- **连锁反应**
    - 雷曼倒闭后，货币基金恐慌性抛售非国债资产 → 美元流动性枯竭
    - 欧洲影子银行无法续借短期美元债务（无法直接求助美联储）

![image.png](https://s1.vika.cn/space/2025/06/21/5309629154e848f4becfe76b373b109c)

- **急救方案：央行货币互换**
    - **第一步**：美联储与欧央行互换货币（美元换欧元）
    - **第二步**：欧央行用这些美元向本国银行放贷
    - **第三步**：美国财政部增发国债吸收回流美元，稳定货币基金
# 危机后

![image.png](https://s1.vika.cn/space/2025/06/21/13c97c533fe94943b006cb4dbd451b5a)

- **临时合作掩盖的根本问题**
    - **金融全球化 vs 国家化监管**：危机最终靠"资产回巢"（MBS由美联储接盘）解决，违背资本自由流动原则
    - **美元困境**：全球储备货币的本质仍是美国国家货币，美联储不可能无限承担国际最后贷款人角色
- **历史重演的风险**
    - 1930年代大萧条正是始于全球化逆转
    - 当前各国政策各自为战（财政刺激/货币宽松），反而加剧债务脆弱性


# **学生提问：美联储对外放贷是否风险过大？**

- **伯南克的辩护**
    - 货币互换以"等量反向平仓"为原则，汇率波动不影响账面，双方各自用自己的货币做账就不会有问题
    - 本质上是以央行信用阻止系统性崩溃
- **现实困境**
    - 国会难以理解这种"无风险"操作（政治敏感性极高）
    - 五大央行（C5）虽建立永久互换协议，但合作上限存疑


# 蒙代尔的警示

央行合作只能暂缓危机，真正解决方案仍需：

1. **重建国际货币纪律**（但缺乏政治共识）
2. **面对深层矛盾**：国家主权与全球化金融的不可调和性
