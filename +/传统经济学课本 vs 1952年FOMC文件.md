| **对比维度**       | **传统经济学课本逻辑**                                                                 | **1952年文件逻辑**                                                                 |
|---------------------|---------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------|
| **核心传导机制**    | 线性链条：利率→银行信贷→实体经济                                                      | 非线性链条：利率→交易商行为→债券市场流动性→资产价格→实体经济                       |
| **政策工具重点**    | 数量工具（准备金、货币供给）                                                          | 价格工具（市场流动性溢价、期限溢价）                                               |
| **金融市场作用**    | 忽略或简化，假设利率直接传导至实体经济                                                | 核心环节，通过债券/股票市场流动性变化放大政策效果                                  |
| **银行角色**        | 被动中介，按货币乘数规则放贷                                                          | 不强调银行，聚焦杠杆化交易商的套利行为                                             |
| **利率形成机制**    | 短期利率→长期利率（通过预期理论）                                                     | 短期利率→交易商套利→流动性溢价变化→长期利率                                        |
| **货币创造逻辑**    | 基础货币→银行信贷扩张→货币乘数效应                                                    | 市场操作→改变资产流动性→私人部门资产负债表调整→信用条件变化                        |
| **政策有效性前提**  | 依赖银行体系正常运转                                                                  | 依赖金融市场深度和交易商行为                                                       |
| **典型模型**        | IS-LM模型、货币乘数公式                                                              | 市场流动性框架、期限溢价模型                                                       |
| **现实案例适用性**  | 适用于20世纪银行主导的金融体系                                                        | 解释2008年QE、2020年公司债购买等现代操作更有效                                     |
| **对杠杆的考量**    | 忽略                                                                                  | 关键因素（如回购融资放大政策效果）                                                 |
| **政策目标侧重**    | 控制货币供应量或短期利率                                                              | 管理市场流动性和风险偏好                                                           |

### **补充说明：简化与现实的冲突点**
1. **货币乘数失效**  
   - 课本假设：1美元准备金→N美元贷款（固定乘数）  
   - 现实：2008年后超额准备金激增，但贷款未同比增加，因银行受资本约束且市场融资渠道多元化。

2. **利率传导断裂**  
   - 课本假设：美联储降息→银行贷款利率同步下降  
   - 现实：2023年美联储加息时，银行存款利率调整滞后，因银行依赖存款粘性而非准备金成本。

3. **非银行金融的崛起**  
   - 课本聚焦商业银行，但2020年美国企业债发行中，影子银行和机构投资者占比超70%，传统信贷渠道解释力下降。

4. **操作规模悖论**  
   - 课本逻辑：大规模"印钞"才能影响利率  
   - 1952年逻辑：小规模精准操作可通过市场预期和套利行为放大效果（如"前瞻指引"替代实际购买）。

这一对比揭示了主流宏观经济学教学与实际政策执行之间的深层鸿沟。
