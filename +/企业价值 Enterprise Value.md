# PE估值的局限性

![image.png](https://s1.vika.cn/space/2025/07/17/b322874b92d240ef96bcf0827490911b)
[在线表格](https://rk7nrn34nu.feishu.cn/wiki/ISZWwUpL1iCcx6ktabWcA5lpn5e?from=from_copylink&sheet=IxZwtq)

- **案例回顾**：两家相同披萨店（相同资产、相同营业利润3万美元），因资本结构不同导致PE估值扭曲：
    
    - **保守型（A）**：全股权出资，市值21万美元（PE=10）。
    - **杠杆型（B）**：1万股权+10万债务，市值18.9万美元（PE=10），但隐含资产价值虚高至27.9万美元。
        
- **矛盾点**：相同资产因融资方式不同被市场赋予不同价值，PE比率失效。

> [!important] 
> **EV = 市值 + 债务 - 现金**（仅扣除非经营性现金）

- 公司A EV 200k
- 公司B EV 190k
- **！！！实际上也就便宜了 5%**


# 使用 EV 反向评估股价


1. **逻辑**：
    
    - 直接对标核心营业利润（EBIT），忽略利息、税项等资本结构差异。
    - **公式**：EV ÷ 营业利润（EBIT）。若行业倍数=6：
        - 两家店营业利润均为3万 → **合理EV=18万美元**。
            
2. **股价重估**：
    - **A公司**：EV=18万 → 股价=18万/1万股=**18美元**
    - **B公司**：EV=18万 → 市值=EV+现金-债务=18+1-10=**9万美元** → 股价=**9美元**
