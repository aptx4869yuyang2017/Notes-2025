#### **1. 核心概念**  
- **远期价格**：合约约定的未来交易价格，需满足**无套利条件**。  
- **现货价格**（$S_0$）：当前资产价格（如黄金 \$1000/盎司）。  
- **持有成本**（$C$）：储存、保险等费用（如 \$50）。  
- **利率差异**：  
  - **无风险利率**（$r_f$）：国债等收益（如 5%）。  
  - **借款利率**（$r_b$）：融资成本（如 10%，通常 $r_b \geq r_f$）。  

---

#### **2. 合理边界公式**  
| **边界类型** | **公式**                               | **逻辑**                                         |
| -------- | ------------------------------------ | ---------------------------------------------- |
| **上限**   | $F_{\text{max}} = S_0 (1 + r_b) + C$ | 若 $F > F_{\text{max}}$，借\$买现货+卖远期套利（如 \$1150）。 |
| **下限**   | $F_{\text{min}} = S_0 (1 + r_f) + C$ | 若 $F < F_{\text{min}}$，买债券+买远期套利（如 \$1100）。    |

**示例**（$S_0 = \$1000$, $C = \$50$）：  
- 上限：$1000 \times 1.10 + 50 = \$1150$  
- 下限：$1000 \times 1.05 + 50 = \$1100$  

---

#### **3. 关键结论**  
1. **无套利区间**：$F \in [F_{\text{min}}, F_{\text{max}}]$。  
2. **均衡价格**：若 $r_b = r_f$，则 $F = S_0 (1 + r) + C$（唯一合理价）。  
3. **市场影响**：  
   - 远期价过高 → 现货需求↑ → $S_0$↑ → $F$↓。  
   - 远期价过低 → 远期需求↑ → $F$↑。  

---

#### **4. 记忆技巧**  
- **上限**：与**借款**相关（成本更高，故价格上限更高）。  
- **下限**：与**无风险投资**相关（收益更低，故价格下限更低）。  
- **口诀**：  
  > “借款拉高上限，债券托住下限，持有成本两边加。”  

--- 

**用途**：快速判断远期合约定价是否合理，识别套利机会。
