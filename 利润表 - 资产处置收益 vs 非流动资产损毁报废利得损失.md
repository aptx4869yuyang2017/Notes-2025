# 关键区分：资产处置收益 vs. 报废利得**


这是核心易错点，区分关键在于 **“主动出售”** 还是 **“被动报废”**。

| 特征       | **资产处置收益 (Asset Disposal Income)** | **非流动资产毁损报废利得/损失**    |
| :------- | :--------------------------------- | :-------------------- |
| **处置意图** | **主动出售、转让**（有交易目的）                 | **被动报废、丢弃**（资产已无价值）   |
| **资产状态** | 资产**仍具有使用/转让价值**（如旧设备）             | 资产**已丧失使用价值**（如废铁）    |
| **会计科目** | 计入 **`营业利润`** 之前                   | 计入 **`营业外收支`**，影响利润总额 |
| **例子**   | 出售闲置设备、转让土地使用权                     | 设备提足折旧后当废品卖、自然灾害导致毁损  |
|          |                                    |                       |

**决策口诀：**
**能卖的 → 资产处置收益（进营业利润）**
**当废品卖的 → 营业外收支（进利润总额）**
