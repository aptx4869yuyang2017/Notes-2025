#### **1. 基本概念**  
- **定义**：401(k)是美国雇主提供的**税前供款**退休储蓄计划，功能类似传统IRA但额度更高。  
- **核心特点**：  
  - **税前供款**：工资直接扣除，降低当期应税收入；  
  - **延后纳税**：账户内投资收益免税增长，取出时按普通收入计税；  
  - **雇主匹配**：部分公司提供"1:1匹配"等额外供款（如员工存$1，雇主补$1）。  

#### **2. 关键规则**  
| **项目**           | **401(k)**            | **传统IRA**           |     |
| ---------------- | --------------------- | ------------------- | --- |
| **2023年供款上限**    | $22,500（50岁以上$30,000） | $6,500（50岁以上$7,500） |     |
| **供款来源**         | 雇主代扣工资                | 个人自主存入              |     |
| **投资选择**         | 雇主指定（如公司股票/基金）        | 自由选择（股票/债券/ETF等）    |     |
| **提前取款（59.5岁前）** | 缴税+10%罚金（特殊困难可豁免）     | 同左                  |     |
| **强制取款年龄**       | 72岁开始（RMDs）           | 同左                  |     |

#### **3. 核心优势**  
- **高供款额度**：年度上限是传统IRA的3-4倍；  
- **雇主匹配**：相当于"免费资金"，加速储蓄增长；  
- **自动化存钱**：工资自动扣除，减少消费诱惑；  
- **紧急借贷**：允许免罚金借款（需偿还本金+利息）。  

#### **4. 适用场景**  
- **优先选择401(k)**：  
  - 雇主提供匹配（如匹配50%供款）；  
  - 高收入人群需更大抵税空间。  
- **搭配传统IRA**：  
  - 401(k)供款已达上限；  
  - 希望自主投资（如个股/加密货币）。  

#### **5. 税务策略**  
- **延税价值**：  
  - 工作期高税率（如32%）→ 退休后低税率（如22%）时更划算；  
  - 投资收益复利增长免税。  
- **风险提示**：  
  - 未来税率可能上涨，导致退休后实际税负增加。  

**一句话总结**：401(k)是"高额度+雇主福利版"的传统IRA，适合最大化利用雇主匹配的长期储蓄工具。
