**📍时间地点**：1999年11月30日，美国西雅图
**📍事件本质**：全球化批评者首次大规模联合反对以“华盛顿共识”为核心的全球自由市场秩序。

---

### 📍一、抗议背景

* WTO召开新一轮多边贸易谈判（“千年回合”）
* 华盛顿共识全球推广：自由贸易、私有化、去管制
* 多方质疑：贫富差距扩大、劳工权益受损、环境被牺牲

---

### 📍二、主要参与者与诉求

| 群体     | 主要诉求               |
| ------ | ------------------ |
| 工会组织   | 反对产业转移、工资压低        |
| 环保团体   | WTO规则忽视环保          |
| 发展援助组织 | 发展中国家被迫开放市场、加剧贫困   |
| 反资本主义者 | 抨击跨国公司控制全球规则       |
| 民主倡导者  | 批评WTO决策不透明、无民主监督机制 |

---

### 📍三、历史意义

#### ✅ **全球化批评运动的起点**

* 全球首次大规模**反自由市场全球化抗议**
* 推动对“全球化等于进步”的质疑进入主流

#### ✅ **挑战 [[华盛顿共识]]**

* 抗议者质疑自由化、私有化、去管制等政策对发展中国家的负面影响
* 成为全球反对华盛顿共识的重要象征事件

#### ✅ **迫使国际组织反思与转型**

* WTO谈判破裂，会议失败
* IMF、世行推动“后华盛顿共识”改革方向（如减贫、制度建设）

#### ✅ **奠定“另一个全球化”路径**

* 催生世界社会论坛（WSF）、全球民间社会网络
* 强调：**以人和环境为中心的全球治理替代方案**

---

### 📍四、关键词理解

| 概念       | 含义                     |
| -------- | ---------------------- |
| 华盛顿共识    | 以自由市场改革为核心的发展政策方案      |
| 新自由主义全球化 | 强调市场主导、贸易自由化、资本自由流动    |
| 后华盛顿共识   | 更强调社会公平、制度建设、政府角色的改革思路 |
| 另一个全球化   | 倡导公平、可持续、民主参与的全球治理     |

---

### 🧠 一句话总结：

> **西雅图抗议是对全球市场霸权秩序的首次大规模反击，开启了“华盛顿共识”全球主导地位的转折时刻。**
