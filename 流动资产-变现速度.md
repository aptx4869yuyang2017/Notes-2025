| 排名 | 科目 | 变现过程 | 核心特点 |
| :--- | :--- | :--- | :--- |
| **1** | **货币资金** | 无需变现 | **本身就是现金** |
| **2** | **交易性金融资产** | 在公开市场出售 | **秒级/天级变现**，价格波动 |
| **3** | **应收票据** | 贴现或到期承兑 | **银行信用担保**，流动性强 |
| **4** | **应收账款** | 等待客户付款或保理 | **依赖客户信用**，有账期和坏账风险 |
| **5** | **合同资产** | 完成条件后转为应收账款 | **权利有条件**，流动性弱于应收账款 |
| **6** | **预付款项** | 收到货物后再销售 | **已支付现金**，无法直接收回 |
| **7** | **存货** | 出售 -> 产生应收账款 -> 收款 | **变现链条最长**，风险最高 |

**重要提示：**
*   **其他流动资产**：如果包含的是一些预缴税款或短期待摊费用，其流动性通常与**预付款项**类似，甚至更差。
*   **一年内到期的非流动资产**：例如一年内到期的长期应收款，其流动性应根据底层资产的性质来判断，通常可以放在**应收账款**之后的位置。

这个排序是财务分析中评估企业**短期偿债能力**（如计算速动比率、现金比率）的基础。
