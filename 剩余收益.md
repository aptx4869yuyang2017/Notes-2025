### 什么是“剩余收益”？

**剩余收益（Residual Earnings / Residual Income）** 是一个衡量企业在扣除必要报酬后所创造的**超额利润**的指标。

#### 公式：

> 剩余收益 = 实际收益 - （必要报酬率 × 投资额）

- **实际收益**：项目或企业在一个时期内产生的净利润。
- **必要报酬率**：投资者要求的最低回报率（比如10%），也叫资本成本。
- **投资额**：初始投入的资本（账面价值）。

这个指标的核心思想是：  
👉 只有当企业的收益**超过**投资者期望的最低回报时，才真正创造了**额外价值**。

---

### 二、例子说明

假设：

- 初始投资 = $400
- 必要报酬率 = 10%
- 项目只经历一个投资期（即第1年）

#### 情况1：收益为 $40

```text
剩余收益 = $40 - (10% × $400) = $40 - $40 = $0
```

→ 收益刚好等于必要报酬，没有超额利润，**不产生价值增值**。

此时该项目的价值为：

```text
价值 = 账面价值 + 预期剩余收益的现值
     = $400 + ($0 / 1.10) = $400
```

→ 价值等于历史成本，无溢价。

---

#### 情况2：收益为 $48（收入总额$448）

```text
剩余收益 = $48 - (10% × $400) = $48 - $40 = $8
```

→ 产生了 **$8 的超额收益**，也就是“超常收益”或“超额利润”。

此时项目的价值为：

```text
价值 = $400 + ($8 / 1.10) = $400 + $7.27 = $407.27
```

→ 价值高于账面价值，说明该项目**创造了价值增值**。
