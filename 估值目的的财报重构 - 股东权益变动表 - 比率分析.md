[耐克 计算案例](https://my.feishu.cn/wiki/R3STwHySGijxEvkE0OJck9MknSf?from=from_copylink&sheet=CQMoFG)


| 比率名称          | 公式                                                                    | 含义        |
| ------------- | --------------------------------------------------------------------- | --------- |
| 股利发放率         | $\frac{\text{股利}}{\text{综合收益}}$                                       | 现金分红占盈利比例 |
| 总发放率          | $\frac{\text{股利} + \text{回购}}{\text{综合收益}}$                           | 总体股东回报比例  |
| 收益留存率         | $1 - \text{总发放率}$                                                     | 企业保留利润比例  |
| 股利占权益账面价值比重   | $\frac{\text{股利}}{\text{权益账面价值} + \text{股利}}$                         | 股利对权益的影响  |
| 发放总额占权益账面价值比重 | $\frac{\text{股利} + \text{回购}}{\text{权益账面价值} + \text{股利} + \text{回购}}$ | 总支付对权益的影响 |
| 普通股权益报酬率      | $\frac{\text{综合收益}}{\text{平均普通股权益}}$                                  | 股东投资的盈利能力 |
| 新增投资率         | $\frac{\text{与股东交易净额}}{\text{期初权益}}$                                  | 股东是否追加投资  |
| **普通股东权益增长率** | $\text{ROCE} + \text{新增投资率}$                                          | 权益整体增长动力  |

**普通股东权益增长率**
=普通股东权益增长额/期初普通股东权益
​=期初普通股东权益(综合收益+与股东之间的交易净额​)
